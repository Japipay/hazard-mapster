One Last Step: Add Your Google Maps API Key

The maps feature is ready to go, but you need to add your Google Maps API key to make it work. Here's how:

  1. Visit Google Cloud Console
  2. Create a project named "Hazard Mapster"
  3. Enable "Maps JavaScript API"
  4. Create an API key in Credentials
  5. Copy the key
  6. Edit barangay-detail.html line 8
  7. Replace AIzaSyDummy123456789 with your actual API key

For detailed instructions, see QUICK_START.md in your project folder.
