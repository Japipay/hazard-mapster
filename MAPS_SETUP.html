üîë To Make It Work - Get a Google Maps API Key

Step 1: Create a Google Cloud Project

  1. Go to Google Cloud Console
  2. Create a new project or select an existing one

Step 2: Enable Maps API

  1. In the Cloud Console, go to "APIs & Services"
  2. Click "Enable APIs and Services"
  3. Search for "Maps JavaScript API"
  4. Click on it and press "Enable"

Step 3: Create an API Key

  1. Go to "Credentials" in the left menu
  2. Click "Create Credentials" ‚Üí "API Key"
  3. Copy the generated API key

Step 4: Add API Key to Your File

Edit barangay-detail.html and find this line:

<script async defer src="https://maps.googleapis.com/maps/api/js?key=AIzaSyDummy123456789&v=3.55"></script>

Replace AIzaSyDummy123456789 with your actual API key:

<script async defer src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY_HERE&v=3.55"></script>
üí° Tip: For development, you can use an API key without restrictions. For production, restrict it to only accept requests from your domain for security.
